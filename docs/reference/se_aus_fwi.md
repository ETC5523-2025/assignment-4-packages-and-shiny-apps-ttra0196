# Fire weather indices for south-eastern Australia, 1979–2020

Illustrative Fire Weather Index (FWI) and Monthly Severity Rating (MSR)
by year for south-eastern Australia. Replace with processed data when
available.

## Usage

``` r
se_aus_fwi
```

## Format

A data frame with 42 rows and 3 variables:

- year:

  integer, 1979–2020

- fwi:

  numeric, Fire Weather Index

- msr:

  numeric, Monthly Severity Rating

## Source

Adapted conceptually from van Oldenborgh et al. (2021).
